O que é SlackCON?

SlackCON é um esforço da comunidade para organizar um evento anual de quatro dias, direcionada a distribuição Slackware Linux, a distribuição Linux mais antiga ainda em desenvolvimento.

O evento é um campo de treinamento, serve para unir mais a comunidade e descobrimentos e claro, ajudar no desenvolvimento do projeto como:

  • Divulgação
  • Publicidade
  • Ideias
  • Bug report
  • Documentação
  • Mantenedor Slackbuilds
  • Comercial
  • Doação direta para o projeto Slackware
  • Empacotamento para comunindade
  • Tecnologias para o mundo GNU/Linux em geral

Com isto conseguimos manter uma comunidade ainda mais fiel e semear o conhecimento do projeto para os que não conhecem.

A ideia inicial é ir variando anualmente os estados do Brasil, ou seja, todo ano o evento é organizado em um estado diferente no Brasil e quem sabe um dia no mundo.
A votação das cidades são feitas pela comunidade, qualquer pessoa pode votar.

O evento inicialmente não é adoçado pelo projeto Slackware, como dito é um esforço da comunidade, não somos também uma empresa. Tudo é feito de uma forma colaborativa pela comunindade. SEM FINS LUCRATIVOS.
